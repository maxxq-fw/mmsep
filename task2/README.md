 «Моделирование онтологий» // Лаб. 2 // 29.10.25
```
Онтология имеет класс Человек, который содержит в себе следующие сущности:  
  
    • Прадед  
    • Прадед_2  
    • Дед
    • Дед_2  
    • Отец  
    • Отец_2  
    • Сын  
    • Сын_2  
  
И следующие свойства:  
    • имеет родителя (inverse of является предком)  
    • является потомком (Transitive + SuperProperty of имеет родителя)  
    • является предком (inverse of имеет родителя)  
  
Пример работы:  
Задав сущности Сын свойство имеет родителя Отец - reasoner установит следующие свойства: является потомком Прадед, является потомком Дед, является потомком Отец;  
Задав сущности Отец свойство имеет родителя Дед - reasoner установит следующие свойства: являестся потомком Прадед, является потомком Дед, является предком Сын;  
```
